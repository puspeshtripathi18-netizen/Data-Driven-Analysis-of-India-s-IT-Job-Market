# 📊 Data-Driven Analysis of India's IT Job Market

## 🚀 Project Overview
This project explores the Indian IT job market using job listing data to uncover trends in salaries, skills, and hiring demand. The analysis was performed using Microsoft Excel and presented through interactive dashboards and visualizations.

## 🎯 Objectives
- Analyze salary trends across IT roles.
- Identify the most in-demand technical skills.
- Study the relationship between experience and salary.
- Understand hiring patterns in the Indian IT industry.
- Create a dashboard for easy data visualization.

## 🛠️ Tools Used
- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Charts & Graphs
- Dashboard Design

## 📈 Key Insights
- Senior-level positions offer higher salaries.
- Python, AI/ML, Data Science, and Cloud Computing are highly sought-after skills.
- Software Development and Data Analytics roles show strong hiring demand.
- Experience and specialized skills significantly influence compensation.

## 📂 Dataset
The dataset contains IT job listings with information such as:
- Job Title
- Salary
- Experience Level
- Required Skills
- Job Category

## 📊 Dashboard Features
- Salary Analysis
- Skill Demand Analysis
- Experience-Level Comparison
- Hiring Trend Visualization

## ✅ Conclusion
This project provides valuable insights into India's IT job market, helping students, job seekers, and professionals understand industry trends and make informed career decisions.
